0/7 days

Day 1: HTML Document Structure & Basic Elements

2-3 hours (Morning: 1.5h theory, Evening: 1h practice)

Beginner
Objectives:
  • Study HTML5 semantic elements: header, nav, main, section, article, aside, footer
  • Learn DOCTYPE declaration, meta tags, character encoding
  • Practice 20 basic HTML elements with proper nesting
Deliverable:

Single-page HTML document with personal bio

Assessment:

Create valid HTML that passes W3C validator

2-3 hours

Beginner

Contact form + skills table integrated into profile page
